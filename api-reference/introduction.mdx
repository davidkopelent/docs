## Welcome

This documentation covers the Checks API, which provides endpoints for retrieving and processing payments for checks at merchant places and desks. The API is built using OpenAPI 3.0 specification and provides comprehensive documentation for all endpoints.

## Authentication

All API endpoints are authenticated using Bearer tokens. Include your JWT token in the Authorization header:

```json
"security": [
  {
    "bearerAuth": []
  }
]
```

## Base URL

All API requests should be made to:

```
https://api.storyous.com
```

## Available Endpoints

- **GET /checks/{merchantPlaceId}/{deskId}** - Retrieve checks for a specific merchant place and desk
- **POST /checks/{merchantPlaceId}/{deskId}** - Process payment for a check

## Response Format

All successful responses return JSON with a consistent structure:

```json
{
  "data": [...],
  "type": "string"
}
```

Error responses include detailed error information to help with debugging.
